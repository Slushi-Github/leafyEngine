# Timers

The ``LfTimer`` class is used to create a timer, with a duration and a callback function.

```haxe
// import the LfTimer class
import leafy.backend.LfTimer;

// create a timer in your state
LfTimer.after(1000, function():Void {
    // do something
});
```
